Pour avoir les signaux moyens, il faut aller chercher la dernière colonne de chaque feuille Excel qui considère une moyenne conditionnelle sur l’ensemble des essais.

 

Exemple : pour le fichier essai9cyl.xlsx / onglet C6

Les 5 premières colonnes correspondent aux essais bruts. Le -14.88 correspond à un moment où le capteur mesure un point sur l’une de ses bornes de mesure. Cette valeur devrait normalement être à 0 ;
Les 5 colonnes suivantes sont utilisées pour définir les écarts temporels existant entre les différents signaux ;
Les 5 colonnes suivantes aux 5 premières colonnes mais remises dans la bonne référence temporelle ;
La dernière colonne (Q) est celle qui considère un signal moyen pour le capteur C6 pour les essais du réseau de 9 cylindres ! C’est celle qui a été utilisée pour représenter les courbes que vous devez avoir vu dans le document PPT ?!!
